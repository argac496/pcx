SELECTOR DASAR CSS
1. Konsep Inti

Selector menentukan elemen HTML mana yang akan dipengaruhi oleh CSS. Jika selector salah atau tidak cocok: tidak ada error, tapi tidak ada perubahan tampilan. Ini adalah konsep paling fundamental di CSS.

1. Selector Elemen (element selector)

1. Konsep Inti

Menargetkan semua elemen dengan nama tag tertentu.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
HTML:
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Selector Elemen</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <p>Paragraf pertama</p>
  <p>Paragraf kedua</p>
</body>
</html>
CSS:
style.css 
p {
  color: blue;
}

Hasil: Semua elemen <p> berwarna biru.

3. Latihan 1 — Selector Terlalu Umum
HTML: 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Selector Elemen</title>
</head>
<body>
  <p>Paragraf pertama</p>
  <p>Paragraf kedua</p>
  <div>Ini div</div>
</body>
</html>
CSS awal: 
p {
  color: red;
}

Tugas: Buat hanya paragraf kedua berwarna merah.

4. Solusi (Kode Benar) 
p {
  color: black;
}

p:nth-child(2) {
  color: red;
}
5. Penjelasan: Selector elemen selalu mengenai semua elemen sejenis. Untuk membedakan elemen yang sama, dibutuhkan selector tambahan.

2. Selector Class (.class)

1. Konsep Inti

Menargetkan elemen berdasarkan atribut class.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Selector Class</title>
</head>
<body>
  <p class="judul">Belajar CSS</p>
  <p>Paragraf biasa</p>
</body>
</html>
style.css 
.judul {
  color: green;
}

Hasil: Hanya teks dengan class judul yang berubah warna.

3. Latihan 2 — Penggunaan Class 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Selector Class</title>
</head>
<body>
  <p class="judul">Belajar CSS</p>
  <p>Paragraf biasa</p>
</body>
</html>
CSS awal: 
.judul {
  color: red;
}

Tugas: Buat teks “Belajar CSS” berwarna merah.

4. Solusi (Kode Benar) 
.judul {
  color: red;
}
5. Penjelasan: Nama class harus sama persis. CSS tidak memberi peringatan jika selector tidak ditemukan.

3. Selector ID (#id)

1. Konsep Inti

Menargetkan satu elemen unik dalam satu halaman.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Selector ID</title>
</head>
<body>
  <div id="header">Header</div>
  <div>Konten</div>
</body>
</html>
style.css 
#header {
  background-color: lightgray;
}

Hasil: Hanya elemen dengan id header yang terkena style.

3. Latihan 3 — Salah Jenis Selector 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Selector ID</title>
</head>
<body>
  <div id="header">Header</div>
</body>
</html>
CSS awal: 
.header {
  background-color: lightblue;
}

Tugas: Beri latar belakang biru muda pada header.

4. Solusi (Kode Benar) 
#header {
  background-color: lightblue;
}
5. Penjelasan: ID menggunakan #, sedangkan class menggunakan .. Keduanya tidak bisa dipertukarkan.

4. Selector Gabungan (element.class)

1. Konsep Inti

Menargetkan elemen dengan tag dan class tertentu sekaligus.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Selector Gabungan</title>
</head>
<body>
  <p class="aktif">Paragraf aktif</p>
  <div class="aktif">Div aktif</div>
</body>
</html>
style.css 
p.aktif {
  color: green;
}

Hasil: Hanya paragraf dengan class aktif yang berubah.

3. Latihan 4 — Target Terlalu Luas 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Selector Gabungan</title>
</head>
<body>
  <p class="aktif">Paragraf aktif</p>
  <div class="aktif">Div aktif</div>
</body>
</html>
CSS awal: 
.aktif {
  color: red;
}

Tugas: Warnai hanya paragraf aktif.

4. Solusi (Kode Benar) 
p.aktif {
  color: red;
}
5. Penjelasan: Selector .aktif terlalu umum. Menggabungkannya dengan tag mempersempit target.

5. Selector Bertingkat (descendant selector)

1. Konsep Inti

Menargetkan elemen berdasarkan struktur HTML.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Selector Bertingkat</title>
</head>
<body>
  <div class="container">
    <p>Paragraf dalam container</p>
  </div>
  <p>Paragraf luar</p>
</body>
</html>
style.css 
.container p {
  color: blue;
}

Hasil: Hanya paragraf di dalam .container yang berubah.

3. Latihan 5 — Struktur Tidak Dipertimbangkan 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Selector Bertingkat</title>
</head>
<body>
  <div class="container">
    <p>Paragraf dalam container</p>
  </div>
  <p>Paragraf luar</p>
</body>
</html>
CSS awal: 
p {
  color: green;
}

Tugas: Warnai hanya paragraf di dalam container.

4. Solusi (Kode Benar) 
.container p {
  color: green;
}
5. Penjelasan: Selector bertingkat membaca hubungan parent-child, bukan hanya jenis elemen.
B
SPECIFICITY (PRIORITAS SELECTOR)
1. Konsep Inti

Jika beberapa selector menarget elemen yang sama, browser memilih aturan dengan prioritas tertinggi, bukan yang ditulis lebih dulu. Urutan dasar: ID > Class > Element.

6. Prioritas Selector

1. Konsep Inti

Browser menggunakan bobot poin untuk menentukan gaya mana yang diterapkan saat terjadi konflik.

2. Contoh Kode yang Benar (HTML + CSS Lengkap)
index.html 
<!DOCTYPE html>
<html>
<head>
  <title>Specificity</title>
</head>
<body>
  <p id="teks" class="warna">Halo CSS</p>
</body>
</html>
style.css 
p { color: blue; }
.warna { color: green; }
#teks { color: red; }

Hasil: Teks berwarna merah.

3. Latihan 6 — Konflik Selector 
<!DOCTYPE html>
<html>
<head>
  <title>Latihan Specificity</title>
</head>
<body>
  <p id="judul" class="aktif">Belajar CSS</p>
</body>
</html>
CSS awal: 
p { color: black; }
.aktif { color: blue; }
#judul { color: green; }

Tugas: Tentukan warna akhir teks.

4. Solusi

Warna akhir: color: green;

5. Penjelasan: Selector ID selalu menang jika konflik dengan class dan element.
C
DISPLAY
1. Konsep Inti

display menentukan perilaku dasar elemen dalam layout.

7. display: block

1. Konsep Inti

Memaksa elemen mengambil seluruh lebar yang ada dan turun ke baris baru.

2. Contoh Kode yang Benar
index.html
<!DOCTYPE html>
<html>
<head>
  <title>Display Block</title>
</head>
<body>
  <span>A</span>
  <span>B</span>
</body>
</html>
span {
  display: block;
  background-color: lightblue;
}
3. Latihan 7

Tugas: Buat setiap span berada di baris terpisah.

4. Solusi 
.box {
  display: block;
  background-color: orange;
}
5. Penjelasan: Dengan display: block, elemen yang tadinya inline (seperti span) akan berperilaku seperti block (seperti div).

8. display: inline-block

1. Konsep Inti

Elemen bisa berdampingan tapi tetap bisa diatur ukuran (width/height).

2. Contoh Kode yang Benar 
<!DOCTYPE html>
<html>
<head>
  <title>Inline Block</title>
</head>
<body>
  <div class="box">A</div>
  <div class="box">B</div>
</body>
</html>
.box {
  display: inline-block;
  width: 80px;
  height: 80px;
  background-color: salmon;
}
3. Latihan 8

Tugas: Susun kotak sejajar ke samping.

.box {
  display: inline-block;
  width: 60px;
  height: 60px;
  background-color: green;
}
5. Penjelasan: Div secara default adalah block. inline-block memungkinkannya sejajar horizontal tanpa menghilangkan kemampuan pengaturan dimensi.
D
BOX MODEL
1. Konsep Inti

Setiap elemen HTML dianggap sebagai kotak. Ukurannya dipengaruhi oleh: Content, Padding, Border, dan Margin.

9. padding dan box-sizing

1. Konsep Inti

Padding menambah ruang di dalam kotak. Tanpa box-sizing yang benar, padding akan memperbesar ukuran total kotak.

2. Contoh Kode yang Benar 
<!DOCTYPE html>
<html>
<head>
  <title>Box Model</title>
</head>
<body>
  <div class="box">Konten</div>
</body>
</html>
.box {
  width: 200px;
  padding: 20px;
  box-sizing: border-box;
  background-color: lightgray;
}

Hasil: Lebar visual tetap 200px.

3. Latihan 9

Tugas: Pastikan lebar visual tetap 200px.

.box {
  width: 200px;
  padding: 20px;
  box-sizing: border-box;
  background-color: skyblue;
}
5. Penjelasan: box-sizing: border-box memaksa browser menghitung padding ke dalam lebar yang sudah kita tentukan.
E
POSITION
1. Konsep Inti

position menentukan cara elemen diposisikan dalam dokumen.

10. position: relative

1. Konsep Inti

Menggeser elemen dari posisi aslinya tanpa mengganggu letak elemen lain di sekitarnya.

2. Contoh Kode yang Benar 
<!DOCTYPE html>
<html>
<body>
  <div class="box">Kotak</div>
</body>
</html>
.box {
  position: relative;
  left: 20px;
  background-color: orange;
}
3. Latihan 10

Tugas: Geser kotak 30px ke kanan.

.box {
  position: relative;
  left: 30px;
  background-color: red;
}
5. Penjelasan: Properti left mendorong elemen dari sisi kiri posisi aslinya sejauh nilai yang diberikan.

11. position: absolute

1. Konsep Inti

Meletakkan elemen relatif terhadap parent tercepat yang memiliki posisi (biasanya relative).

2. Contoh Kode yang Benar 
<!DOCTYPE html>
<html>
<body>
  <div class="parent">
    <div class="child">Anak</div>
  </div>
</body>
</html>
.parent {
  position: relative;
  width: 200px; height: 200px;
  background-color: lightgray;
}
.child {
  position: absolute;
  top: 0; right: 0;
  background-color: red;
}
3. Latihan 11

Tugas: Pastikan posisi anak relatif ke parent.

.parent {
  position: relative;
}
5. Penjelasan: Jika parent tidak memiliki position: relative, maka absolute akan merujuk langsung ke layar (body).
F
FLEXBOX (INTI)
1. Konsep Inti

Flexbox mengatur layout anak elemen, bukan elemen tunggal. Ini adalah standar modern untuk layouting.

12. display: flex

1. Konsep Inti

Menyusun item-item di dalamnya secara horizontal (default) dalam satu baris fleksibel.

2. Contoh Kode yang Benar 
<!DOCTYPE html>
<html>
<body>
  <div class="container">
    <div class="item">1</div>
    <div class="item">2</div>
  </div>
</body>
</html>
.container {
  display: flex;
}
.item {
  width: 60px; height: 60px;
  background-color: coral;
}
3. Latihan 12

Tugas: Susun item sejajar.

.container {
  display: flex;
}
5. Penjelasan: Cukup dengan satu baris CSS pada container, semua anak elemennya akan otomatis sejajar ke samping.

13. justify-content, align-items, gap

1. Konsep Inti

Mengatur perataan dan jarak antar item dalam flex container dengan sangat mudah.

2. Contoh Kode yang Benar 
.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}
3. Latihan 13

Tugas: Beri jarak otomatis antar item supaya tidak menempel.

.container {
  display: flex;
  gap: 15px;
}
5. Penjelasan: Properti gap menggantikan peran margin yang seringkali ribet jika digunakan untuk memberi jarak antar kolom.
G
RESPONSIVE DASAR
1. Konsep Inti

Website harus menyesuaikan ukuran layar agar tetap bisa dibaca dengan baik di desktop maupun HP.

14. max-width

1. Konsep Inti

Mencegah elemen melebihi lebar layar tertentu agar tidak muncul scroll horizontal.

2. Contoh Kode yang Benar 
.content {
  max-width: 600px;
  margin: auto;
}
3. Latihan 14

Tugas: Batasi lebar konten agar tidak terlalu lebar di layar besar.

.content {
  max-width: 600px;
}
5. Penjelasan: Penggunaan max-width alih-alih width membuat elemen tsb menciut saat layar lebih kecil dari 600px, tapi tidak membesar melebihi 600px.
