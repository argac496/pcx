Konsep: Memberi Gaya (CSS)

Kalau HTML itu Body, maka CSS itu adalah Pakaian & Hiasan. Rumusnya simpel:

target { gaya: isi; }

Semua gaya harus diapit kurung kurawal { }.

Misi 0: Ubah Warna Tulisan!

MISI PRAKTIK 💻

Ketiklah perintah CSS ini:

p { color: red; }
Tekst Paragraf Jadi Merah!
MISI DETEKTIF 🕵️‍♂️

Ada yang lupa! Penutup titik-koma (;) itu wajib.

h1 { color: blue }
h1 { color: blue; }
HASIL BIRU

Konsep: Warna & Latar

Gunakan color untuk huruf, dan background-color untuk cat dindingnya.

color vs background

Misi 1: Warnai Kotak!

MISI PRAKTIK 💻

Ketiklah perintah background ini:

div { background-color: yellow; }
KOTAK KUNING
MISI DETEKTIF 🕵️‍♂️

Namanya salah! Gak ada property ini.

p { color-text: green; }
p { color: green; }
Teks Jadi Hijau

Konsep: Mengatur Ukuran

Mau tulisan Raksasa atau Kecil? Pakai font-size dengan satuan px.

font-size: 50px;

Misi 2: Bikin Tulisan Gede

MISI PRAKTIK 💻

Ketiklah ukuran font ini:

h1 { font-size: 80px; }
RAKSASA!
MISI DETEKTIF 🕵️‍♂️

Hati-hati, lupa kasih satuan "px"!

p { font-size: 30; }
p { font-size: 30px; }
Ukuran 30 Pixel

Konsep: Jarak Dalam & Bingkai

Padding adalah jarak udara di dalam kotak. Border adalah garis bingkainya.

padding & border

Misi 3: Buat Bingkai Tebal

MISI PRAKTIK 💻

Ketiklah gaya bingkai ini:

div { border: 10px solid black; }
BINGKAI TEBAL
MISI DETEKTIF 🕵️‍♂️

Tanda titik-duanya tertukar!

p { padding; 40px: }
p { padding: 40px; }
ISI BERJARAK

Konsep: Jarak Luar & Sudut

Margin adalah menjauhkan kotak dari tetangga. Border-radius untuk membulatkan sudut.

radius = Bulat

Misi 4: Bikin Lingkaran!

MISI PRAKTIK 💻

Ketiklah angka bulat ini:

div { border: 1px solid red; border-radius: 50px; }
BULAT
MISI DETEKTIF 🕵️‍♂️

Typo di nama property! Gak jalan deh.

div { border-bunder: 20px; }
div { border-radius: 20px; }
LENGKUNG
